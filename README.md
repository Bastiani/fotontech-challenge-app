### FotonTech chellenge app

## install and run

- `yarn`
- `yarn update-schema` with server run
- `yarn relay`
- `yarn start` or `react-native run-ios --simulator="iPhone 6"` or `react-native run-android`
- release test `react-native run-android --variant=release`
